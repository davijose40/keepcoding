Create React App

Pode ser instalado o pacote create-react-app globamente e a partir desse pacote criar seu projeto usando npm create-react-app

Apesar de ser possível a recomendação oficial é que a partir da versão npm 5.2 seja desistalado o pacote global do create-react-app e use npx para iniciar seu projeto React.

Configurando Projeto

Você não precisa instalar ou configurar ferramentas extras usando o comando npx create-react-app basta rodar o comando e construir seu projeto.

Criar novo projeto

npx

npx crate-react-app my-app

npm

npm init react-app my-app

Yarn

yarn create react-app my-app

yarn create disponível a partir Yarn 0.25+

Usando Template

Opcionalmente pode-se usar um template adicionando --template [template-name] ao comando de criação.

em construção

Creating a TypeScript app

Você pode criar um novo projeto TypeScript usando template. Para usar nosso TypeScript template acrescente --template typescript no comando de criação.

npx crate-react-app my-app --template typescript

Quando você cria um app usando CLI o pacote create-react-app vai usar Yarn para instalar as dependencias por padrão se o Yarn estiver instalado mas se ainda assim houver a preferencia ou necessidade de usar o npm só acrescenter ao comando a flag:

npx create-react-app my-app --template typescript
